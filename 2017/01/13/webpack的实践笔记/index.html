在此说明两点：

1、 npm的start是一个特殊的脚本名称，在命令行中使用npm start就可以执行相关命令，如果对应的此脚本名称不是start，想要在命令行中运行时，需要这样用npm run {script name}，如npm run dev.
2、 window环境下不支持&连接命令执行，如：gulp & nodemon mock-server.js。
